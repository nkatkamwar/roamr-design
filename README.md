## What this is

A click-through, single-device prototype built from the approved Roamr v1 mockups.
Six navigable screens with working flows — no backend (design prototype):

1. **Onboarding** · first capture (empty state + install prompt)
2. **Capture flow** (paste → enrichment → mandatory location-add)
3. **Places library** (category/city browse, source label on every card)
4. **Itinerary generation** (cloud-consent sheet: once / always / no)
5. **Itinerary · day plan** (morning/afternoon/evening + egress honesty note)
6. **Settings · privacy** (consent revoke + egress ledger)

## How to navigate

- Tap the highlighted controls inside the phone to move through flows
- Use the numbered chips at the top, the ◂ ▸ arrows, or keyboard (1–6 / ← →)
- Deep-link any screen with `#1`…`#6`

Installable as a PWA (Add to Home Screen) on iOS Safari.
